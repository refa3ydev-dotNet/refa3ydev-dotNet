My main focus is backend development using **C#**, **.NET / ASP.NET**, and **SQL Server**, while also building modern frontend interfaces with **React**, **TypeScript**, and **Tailwind CSS**.

I care about architecture, code quality, debugging, performance, and writing solutions that stay clean as projects evolve.
